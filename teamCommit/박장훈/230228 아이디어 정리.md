# 어린아이 감정 교육
## 상세 내용

- **대상** : 5세 ~ 8세 아이 + 부모님
- 감정 교육 + 한글 교육

### 동화

- 그림 + 글씨 + 소리(읽어주기)
- 페이지 넘어가기 전에 아이에게 상호작용 요구
    - 감정 표현(주인공에게 공감)
    - 특정 물체 그려보기 → 그림 인식 후 해당 글자 써보는 연습
- 상황에 맞는 감정을 이해해보기?
    - 해당 상황에 맞는 감정을 말해보게 하고, 직접 따라하게 하기?
- 북마크 기능 (아이 집중력 고려)

### 독서일기

- 그림 베이스로 기록
    - 다이어리 형식으로 날짜별로 정리
    - 그림을 바탕으로 감성분석 =]
